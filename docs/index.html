Overview

This Rhino plugin adds a contextual radial menu around your cursor, so common commands are always close to where you work. You can set up to 3 submenu levels. Compatible with macOS and Windows.

Main Features

Interactions

  • Drag and drop Rhino toolbar items to add menu items to the radial menu.
  • Rearrange radial menu items with drag and drop.

Execution

  • Launch Primary and Alternate Rhino commands with left and right click.

Configuration

  • Manage multiple menu setups with smart rules based on the current Rhino selection.
  • Edit menu items manually when needed: icon, primary or alternate command, and tooltip.
  • Customize the radial menu in the settings editor: theme, animation speed, configuration rules, and shape.

Key Benefits

  • Faster command access with less pointer travel across distant toolbars.
  • Better viewport focus with fewer interruptions between modeling and interface controls.
  • More fluid workflows by grouping actions in a clear circular layout.

Notes

Current Limitations

  • No keyboard navigation or hotkeys yet.
  • Custom shortcuts using a modifier key and mouse button are not yet supported.
  • Smart Config is functional, but it will be improved over time.

Beta Demo Version

  • The final commercial release is coming soon.
  • All features are enabled in the beta demo version.
  • Settings are not persisted.
  • When Rhino restarts, the radial menu returns to the default setup.

Add the RadialMenu command to your favorite shortcut, whether it is a mouse button, a SpaceMouse button, or a keyboard shortcut, then start modeling.

How to Use

1. Show or Hide the Radial Menu

  • Run the RadialMenu command to show the menu.
  • If the menu is already visible, run RadialMenu again to hide it.
  • You can assign RadialMenu to any keyboard shortcut or mouse button in Rhino settings.
OS Run primary action Run alternate action Drag and drop a menu item Settings Edit menu item
macOS Left click (no modifier) Right click (no modifier) Left click + Cmd key In the center circle, right click + Cmd key On a menu item, right click + Cmd key
Windows Left click (no modifier) Right click (no modifier) Left click + Alt key In the center circle, right click + Control key On a menu item, right click + Control key

If no modifier is listed, use click only.
